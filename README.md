# NYC Traffic Crash Data Explorer
 
This interactive web map visualizes traffic crash data in New York City, focusing on pedestrian and cyclist injuries and fatalities. Built with Mapbox GL JS, the tool allows users to filter crash data by year, victim type, and contributing factors. 


# Features
1. Year (2012-2025)
2. Victim type (Pedestrian, Cylclist)
3. Contributing factors 
4. Click on map markers to display detailed crash information in a styled popup
5. Reset filters to return to full dataset view
